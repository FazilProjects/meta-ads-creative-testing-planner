# AGENTS.md

## Project Context

This repository contains a static portfolio dashboard called Meta Ads Creative Testing Planner.

It is for Fazil Waseem, an aspiring Performance Marketer learning Google Ads, Meta Ads, lead generation, ecommerce marketing, and AI-assisted marketing systems.

## Core Rule

Keep the project honest: this is a creative testing and campaign planning dashboard, not a real Meta Ads platform.

Do not add:

- Meta Ads API integrations
- Real ad account authentication
- Claims of official Meta affiliation
- Guaranteed performance claims
- External framework dependencies

## Technology

- HTML
- CSS
- JavaScript
- No external frameworks or libraries
- GitHub Pages compatible

## Frontend Standards

- Maintain a clean, premium dashboard UI.
- Use a light background with dark text.
- Use Fazil's accent colors carefully: blue, red, yellow, and green.
- Keep animations subtle and professional.
- Respect `prefers-reduced-motion`.
- Preserve responsive behavior across desktop, tablet, and mobile.
- Prevent horizontal overflow on small screens.
- Wrap wide tables in responsive scroll containers.

## Content Standards

Use practical performance marketing language while keeping claims realistic.

The tool may suggest:

- Creative angles
- Hooks
- Primary text
- Headlines
- CTA ideas
- Budget test plans
- Winning ad criteria

The tool must not imply:

- It connects to Meta
- It launches ads
- It guarantees winners
- It reads live performance data

## File Map

- `index.html` - main dashboard structure
- `style.css` - visual system and responsive styles
- `script.js` - local planning logic
- `README.md` - project overview
- `docs/PROJECT_BRIEF.md` - goals and scope
- `docs/DESIGN_SYSTEM.md` - design direction
- `docs/CONTENT.md` - copy and claims guidance

## Change Guidance

When improving the project:

- Keep edits focused.
- Prefer readable, simple code over heavy abstractions.
- Test at mobile, tablet, and desktop widths.
- Keep all generated content local and deterministic unless the project scope changes explicitly.
