# Meta Ads Creative Testing Planner

A professional frontend dashboard portfolio project by Fazil Waseem.

This project helps users turn a short business brief into a structured Meta Ads creative testing plan. It is built with simple HTML, CSS, and JavaScript, with no external frameworks, APIs, or platform integrations.

## Purpose

The planner is designed as a portfolio project for an aspiring Performance Marketer learning Google Ads, Meta Ads, lead generation, ecommerce marketing, and AI-assisted marketing systems.

It demonstrates how a business brief can be translated into:

- Campaign objective direction
- Audience segment planning
- Offer and product/service framing
- Creative angles
- Hook ideas
- Primary text variations
- Headline variations
- CTA suggestions
- Creative testing matrix
- Budget testing plan
- Winning ad checklist
- Exportable test plan

## Important Positioning

This is not a real Meta Ads platform and does not connect to Meta Ads, APIs, ad accounts, or external tools. It is a local planning dashboard for creative testing strategy and portfolio presentation.

## Files

- `index.html` - dashboard markup and planner sections
- `style.css` - responsive premium dashboard styling
- `script.js` - local creative planning logic, export, and reset behavior
- `docs/PROJECT_BRIEF.md` - project scope and goals
- `docs/DESIGN_SYSTEM.md` - visual direction and responsive rules
- `docs/CONTENT.md` - content model and honest-claims guidance
- `AGENTS.md` - contributor and AI-agent instructions

## Run Locally

Open `index.html` in a browser. The project is GitHub Pages compatible because it uses static files only.

## Responsive Requirements

The layout uses a viewport meta tag, global `box-sizing: border-box`, responsive grids, mobile stacking, scroll-safe tables, and no fixed widths that should cause horizontal overflow on small screens.
