# Project Brief

## Project Name

Meta Ads Creative Testing Planner

## Project Goal

Build a professional frontend dashboard that helps users turn a short business brief into a practical creative testing plan for Meta Ads-style campaign planning.

The tool should support planning around:

- Campaign objective
- Target audience
- Audience segment
- Offer
- Product or service
- Pain point
- Promise or outcome
- Creative angle
- Hook
- Primary text
- CTA
- Budget test plan
- Winning ad criteria

## Portfolio Positioning

This is a portfolio project for Fazil Waseem, an aspiring Performance Marketer learning:

- Google Ads
- Meta Ads
- Lead generation
- Ecommerce marketing
- AI-assisted marketing systems

The project should show strategic thinking, structured planning, and frontend execution without pretending to be an official ad platform.

## Product Scope

### Included

- Business brief input
- Campaign objective selector
- Audience segment planner
- Offer and product/service input
- Creative angle generator
- Hook ideas
- Primary text variations
- Headline variations
- CTA suggestions
- Creative testing matrix
- Budget testing plan
- Winning ad checklist
- Export creative test plan
- Reset functionality
- Responsive dashboard UI

### Not Included

- Meta Ads API connection
- Login or authentication
- Real campaign creation
- Real ad account data
- Automated budget recommendations based on live performance
- Claims of guaranteed results

## Success Criteria

- The first screen clearly communicates a professional creative planning dashboard.
- Users can complete the form and generate a readable test plan.
- The output is specific enough to feel useful, while remaining honest and educational.
- The layout works on desktop, tablet, and mobile.
- Tables are scroll-safe on small screens.
- The project can be published through GitHub Pages.

## Build Constraints

- Use plain HTML, CSS, and JavaScript.
- Use no external frameworks or libraries.
- Keep assets and logic local.
- Respect `prefers-reduced-motion`.
- Keep the dashboard lightweight and easy to review.
